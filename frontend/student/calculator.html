How Forecasts Work

  • Total Periods: The semester duration limit (hours) of this subject. You can edit this field for any subject in the list.
  • Safe Absences: The maximum remaining classes you can miss to still finish the semester with at least the target percentage.
  • Classes Needed: The minimum remaining classes you must attend to secure the target. If the remaining classes are too few, it marks it as Impossible.
